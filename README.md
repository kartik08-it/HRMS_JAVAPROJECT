# HRMS - Human Resource Management System

A Java-based Human Resource Management System for managing employees, attendance, leave requests, and payroll operations.

## Features
- Employee information management
- Attendance tracking
- Leave management system
- Payroll processing
- Department and role management

## Technologies
- Java
- MySQL/Database
- JDBC/Hibernate
- Swing/JavaFX (if applicable)

## Setup
1. Clone the repository
2. Configure database connection in `config` file
3. Run the application

## Usage
Launch the application and log in with admin credentials to manage HR operations.
